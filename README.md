# GilDelta

Dalamud plugin (API 15) that tracks gil flow across your own wallet, all
retainers, and the FC chest. Categorizes each change (submarine return,
retainer sale, NPC purchase, marketboard buy, repair, teleport, …) and
surfaces the data through a configurable widget and a dashboard.

Read-only — never writes back to game state.

Full documentation lands in Plan 4 (polish + release).
